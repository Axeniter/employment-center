# Центр занятости населения
## Описание
Веб сервис для поиска работы
## Структура проекта
Папки:
`server` - сервер приложения, написанный в микросервисной архитектуре (в данный момент не работает)
`server-lite` - сервер приложения, написанный в монолитной архитектуре, с помощью фреймворка FastAPI
`app` - клиентское приложение, написанное на фреймворке MAUI для создание кроссплатформенных приложений
## Как использовать
Для начала загрузите репозиторий к себе на компьютер:
`git clone https://github.com/Axeniter/employment-center.git`
### Сервер
Требования:
- Докер
Запуск с терминала:
1. `cd путь_до_репозитория/employment-center/server-lite` - переходим в директорию сервера
2. `copy .env.example .env` - копируем шаблонный файл переменных окружения и выставляем свои настройки
3. `docker-compose up` - запускаем docker контейнер.
